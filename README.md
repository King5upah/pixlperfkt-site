# pixlperfkt-site

Static marketing + legal site for **PixlPerfkt**, served via GitHub Pages at
[pixlperfkt.com](https://pixlperfkt.com).

- `/` — landing page
- `/privacy` — Privacy Policy (App Store *Privacy Policy URL*)
- `/support` — Support page (App Store *Support URL*)

Plain HTML + one stylesheet, no build step. `CNAME` binds the custom domain;
`.nojekyll` skips Jekyll processing.

The app itself lives in a separate private repo.
